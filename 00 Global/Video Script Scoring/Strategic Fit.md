# Strategic Fit

Scores whether the script serves the right persona at the right awareness stage with a consistent angle — and whether those choices trace to the Batch Plan.

---

## What This Scores

A script can be well-written, well-paced, and visually specific — and still be the wrong ad. This dimension evaluates: does the script address the persona's actual pain (not a demographic), use VoC language that persona would recognize, frame the message for the correct awareness stage, and hold a consistent angle from hook through close? Strategic drift is invisible to copy-level scoring — the body sounds good but it's talking to the wrong person, at the wrong awareness level, or it shifts angles mid-script.

## Calibration Table

| Score Range | Description |
|---|---|
| 90-100 | Script addresses the persona's specific pain/trigger (named in Persona Context), not a demographic. At least 2 VoC phrases or buyer-language patterns from Review Analysis appear in the script (sourced). Awareness stage framing is correct — Unaware scripts never name the product before the mechanism; Product-Aware scripts skip problem education. Angle stated in Batch Plan is the angle delivered — hook, body, and close all serve the same argument. No angle drift between sections. |
| 70-89 | Persona addressed by pain but VoC language is generic — could apply to any persona in the brand, not the assigned one. OR: awareness framing is mostly right but 1 section is mismatched (e.g., product named too early for Unaware, or body re-explains the problem for Product-Aware). OR: angle holds through hook and body but close drifts to a different argument. |
| 50-69 | Persona addressed by demographic ("women over 40," "busy moms") instead of pain/trigger. No identifiable VoC language. Awareness stage violated — Unaware script leads with product name, or Most-Aware script spends 20 seconds on problem education. Angle inconsistent — hook sets up one argument, body delivers another. |
| Below 50 | Script could apply to any brand in the category with a find-and-replace on the product name. No persona specificity. No awareness-appropriate framing. Angle unclear or absent. Rewrite from scratch — evaluator must name the specific persona pain, VoC phrases, and awareness framing to use. |

---

## Instant-Fail Triggers

- **Demographic persona naming.** If the script's framing targets a demographic label ("men over 40," "busy professionals") instead of a pain/trigger per Persona Context, auto-caps at 59. Personas are named by what hurts, not who they are.
- **Angle contradiction.** If the hook sets up one argument and the body delivers a contradictory one (hook says "you don't need a doctor," body says "doctor-recommended"), auto-caps at 49.

## Scoring Checklist

### Persona Specificity
- Does the script address the assigned persona's pain/trigger as named in Persona Context? (Not a generic category pain, but THIS persona's version of it.)
- Are at least 2 VoC phrases or buyer-language patterns from Review Analysis present? Name each phrase and its source.
- Would this persona recognize themselves in the first 5 seconds? The hook must name their condition, experience, or trigger — not a demographic.
- Does the script avoid the "any brand" test? Could a competitor use this exact script by swapping the product name? If yes, persona specificity is too low.

### Awareness Stage Alignment
- Check the assigned awareness stage from the Batch Plan. Then verify:
  - **Unaware:** Opens with the problem or an insight. Product name appears late (body or close, not hook). No assumption the viewer knows the category.
  - **Problem-Aware:** Names the problem the viewer already recognizes. Transitions to mechanism/solution. Product can appear in body.
  - **Solution-Aware:** Assumes the viewer knows solutions exist. Differentiates THIS solution. Product appears early.
  - **Product-Aware:** Assumes the viewer knows the product. Focuses on proof, offer, or objection handling. Minimal problem education.
  - **Most-Aware:** Leads with offer, social proof, or urgency. No education needed.
- Does any section violate the stage? (e.g., a Problem-Aware script that never names the problem, or an Unaware script that leads with the product.)

### Angle Consistency
- State the angle from the Batch Plan in one sentence.
- Does the hook set up this angle? (Not a different angle that happens to be related.)
- Does every body beat advance this angle? (No mid-script pivots to a different argument.)
- Does the close resolve this angle? (Not a generic CTA disconnected from the body's argument.)
- If the angle includes a Three Selves target (Actual / Ideal / Ought), does the emotional register match throughout?

---

## Feedback Format

When scoring below 90:
- Which sub-check failed (Persona, Awareness, Angle) and what specifically is wrong
- The assigned persona, awareness stage, and angle from the Batch Plan
- For persona failures: the specific pain/trigger from Persona Context that should be used, plus 2-3 VoC phrases from Review Analysis that should appear in the script
- For awareness failures: which section violates the stage and how to reframe it (e.g., "Hook names the product at 0:03 but this is an Unaware script — move product name to body beat 3 after mechanism is established")
- For angle failures: where the drift starts and a concrete direction to realign (e.g., "Body beat 2 shifts from 'hidden deficiency' angle to 'doctor distrust' angle — rewrite beat 2 VO to connect the deficiency back to the testing gap, not to doctor skepticism")
