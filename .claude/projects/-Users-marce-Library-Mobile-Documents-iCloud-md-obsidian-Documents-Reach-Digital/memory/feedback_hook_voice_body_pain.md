When the strategist drafts hooks for an actor to read on camera, keep them close to the original. Strategist hooks sound like real people talking; rewritten hooks tend to sound more "strategic" and over-engineered.

**Why:** The strategist writes hooks from intuition and real UGC cadence. AI rewrites introduce structure-first language that sounds written, not spoken. The hooks the actor reads on camera must feel natural above all else.

**How to apply:**
- If the strategist provides draft hooks, edit for criteria compliance (fix stats, name the product where "this" is vague, etc.) but preserve the VO phrasing and energy. Don't restructure the thought.
- Captions are editorial and CAN be rewritten for different core types — the actor doesn't read those.
- Body beats in progression timelines should land on the pain point that gets solved at each milestone ("no more morning stiffness," "no more three-day recovery"). The user wants viewers to feel each milestone as a specific pain disappearing, not just a mechanism firing.
- "From the first sip" temporal discounting is a strong opener the strategist values — don't replace it with alternative-callout openers unless asked.
