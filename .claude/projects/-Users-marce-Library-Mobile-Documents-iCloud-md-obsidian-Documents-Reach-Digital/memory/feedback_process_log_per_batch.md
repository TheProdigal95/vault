Process Log starts empty each batch. Do NOT carry it forward from the previous Working Document.

**Why:** The user explicitly corrected the assumption that Process Log is a running record across batches. Each batch's learnings are self-contained.

**How to apply:** When creating a new batch's Working Document, carry forward Script Criteria and Context Files for Agents, but leave Process Log empty.
