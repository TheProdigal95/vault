# Script Structure: Three-Test Investigation

"SCAM OR REAL?" / "I went looking for the catch." The creator is skeptical, tests 3 specific product claims with physical demos, and converts. Each test gets a clear pass/fail verdict with a competitor comparison. The skepticism IS the trust signal — she voices the viewer's objection before they do.

### Core Breakdown
- **Structure:** Skeptical hook (props, price gap, "looking for the catch") → announce 3 tests → test each one with physical demo + competitor comparison → verdict ("all 3 passed") → close.
- **VO register:** Female 30-50, skeptical-turning-convinced. Direct, slightly dry. Debriefing a colleague, not performing.
- **Time to benefit:** ~12s. Tests start immediately after the hook.
- **Rhythm:** 8-10s per test. Each test: name the test → physical demo on product → same test on competitor → reaction → verdict caption.
- **Duration:** 65-80s.
- **Close type:** Guarantee-forward ("feel the difference or get every dollar back").
- **Visual grammar:** Hands-on physical demos — no CGI. Thumb pressing insole, pushing sole on wet tile, spreading fingers in toe box. Split-test format: product passes, competitor fails, side by side. Phone-quality, counter/table surface.
- **Hook defaults:** Overhead counter with props (orthotics receipt, competitor shoe, product box, "SCAM OR REAL?" sticky note). Swollen feet TK direct clip. Foot massage POV.
- **Caption strategy:** TEST 1 / TEST 2 / TEST 3 header cards. ✅ for product passes, ❌ for competitor failures. Two-channel: VO narrates the investigation, captions show verdicts.

### When to Use
- Products priced significantly below the category benchmark ($60 vs $300 orthotics)
- When the viewer's primary objection is "this is too cheap to be real"
- Products with 3+ physically demonstrable features (arch support, grip, flexibility, toe box)
- When the "investigation" framing creates trust that a direct pitch would not

### Key Rules
- Exactly 3 tests — not 2, not 5. Three is the format.
- Each test must be PHYSICAL: thumb pressing foam, pushing sole on wet tile, spreading fingers in toe box. No CGI, no claims-only.
- Each test must include a competitor comparison (product passes, competitor fails on the same test)
- The creator must start skeptical and convert — not start neutral and confirm. The arc is skepticism → surprise → conversion.
- "I went looking for the catch" is the structural premise. The body delivers the investigation. The close reveals: there was no catch.
