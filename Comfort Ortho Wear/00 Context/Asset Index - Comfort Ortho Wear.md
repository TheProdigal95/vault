## Usage Notes

- Use actual Drive links in editor-facing Footage sections; do not link local vault files.
- Blur competitor logos in all footage.
- If competitor brand names appear as on-screen text, mask one letter (`Sk*tchers`, `H*ka`, etc.).
- The underused B-roll shortcuts folder should be included broadly in ComfortWear script Footage/B-Roll sections as an optional asset pool.
