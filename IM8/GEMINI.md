# IM8 Project Instructions

## Hard Rules
- **Background Videos:** ONLY use videos that last exactly 15 seconds as background videos. Do not use shorter loops.
- **Mute All Videos:** Every single `<video>` tag (b-roll and background) MUST include `muted`, `playsinline`, and `data-volume="0"`. No audio should ever be emitted.
- **Typographic Scale for Symbols:** Signs like `$` and `%` in the headline or massive stat lockups MUST be 25% smaller than the main font size it accompanies (e.g., `<span style="font-size: 75%">%</span>`).
- **File Integrity:** Never delete or destroy existing files or folders. Always iterate by creating new versions (e.g., v2, v3) and adding to the canvas.
