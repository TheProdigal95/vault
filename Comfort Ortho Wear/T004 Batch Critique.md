## Promote to Global Criteria

### 1. Skechers foam-collapse cap

**Rule:** No single competitor knock may appear in more than 3 scripts per batch. Vary the specific failure mechanism — same brand, different angle (e.g., "no arch support" vs "foam compresses" vs "not built for standing").

**Basis:** "Do not overuse foam-collapse framing" flagged in T003 Test Log. T004 uses Skechers foam-collapse in 5 scripts. 2 consecutive batches.

**Edit pending** — will add to Video Script Criteria after strategist review of this critique.

### 2. Hook SPM floor for short time windows

**Rule:** Hook rows with time windows of 5 seconds or less must not exceed 280 SPM (~23 syllables per 5s, ~14 per 3s). Write for speaking pace, not reading length. If the hook concept requires more text, extend the time window.

**Basis:** 10 hooks across 6 scripts exceed 300 SPM in this batch, with the worst at 840 SPM. All failures are in 3-6s windows.

**Edit pending.**

### 3. Authority format: product-demo captions must add second-channel info

**Rule:** In authority-format scripts (rating, reframe, comparison), captions in the product-demo section must carry information the VO does not state: clinical terms, comparison data, time thresholds, or mechanism specifics. Truncated VO is not a caption.

**Basis:** Systematic violation across all 7 authority-format scripts in this batch (#1, #2, #3, #10, #11, #12, #13).

**Edit pending.**
