## Cross-Batch Patterns

**1. Caption compression as default fallback.** Writers without explicit cross-script caption audit defaulted to compressing the VO into shorter form for captions — same angle, fewer words — in 10 of 45 hooks reviewed. This is the most consistent caption failure mode across Lifeforce T003, IM8 T002, and Elevate. Root cause: writers write VO first (correct) then reach for a caption and take the path of least resistance. The fix is structural: Caption Compression Ban rule now added to criteria so Tier 1 agents catch it at draft time.

**2. Close CTA structural convergence.** When given the same price anchor ($49) and no explicit cross-script constraint, writers default to the same verb-phrase + trailing-phrase structure. 7 of 15 scripts converged on the same close architecture. Root: the model script's close is strong, writers model it. The fix is the Close CTA Diversity rule now added to criteria — writers must grep for existing close phrasings before writing.

## Promote to Global Criteria

**Rule 1: Caption Compression Ban**
Added to `00 Global/Criteria/Video Script Criteria.md` — under "On-Screen Captions → Caption-vs-VO check" section.
Exact rule: *"A caption that shortens the VO's message into fewer words — without changing the angle or hook type — is a same-angle violation even if no word is shared verbatim. The caption must address a different viewer intent than the VO, not compress the same intent. Test: remove the VO. Does the caption still open its own curiosity gap or land its own payoff, independent of the VO? If the caption only makes sense because of the VO it abbreviates, rewrite it."*
Recurred in 10 hooks this batch across 8 scripts. Prior batches (IM8 T002 caption audit, Elevate) showed same pattern.

**Rule 2: Close CTA Diversity**
Added to `00 Global/Criteria/Video Script Criteria.md` — under "Close (Final 5-10 Seconds)" section.
Exact rule: *"Before writing any close CTA, scan the scripts file for close VO phrasings already used in the batch. No close CTA may share its primary verb phrase (Start/Get/Run/Begin your labs) AND its trailing phrase (link below / the link is below) with any other script in the same batch. Max 2× same structure across a 15-script batch. Differentiators include: different action verb, different product anchor phrase, different reason-to-act frame. The $49 price anchor must still appear — only the surrounding phrasing changes."*
Recurred 7 of 15 scripts this batch. First formal documentation — density of 7/15 warrants immediate promotion rather than waiting for a second batch.
