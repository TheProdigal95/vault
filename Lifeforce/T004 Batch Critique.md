## Cross-Batch Patterns

### Pattern 1 — "Link below" CTA saturation (5/15 scripts)
"Link below" or "link is below" appeared in 5 of 15 scripts as the close VO trailing phrase. This is the highest recurrence of any single CTA construction in this batch. The 2× cap rule in Universal Copy Rules caught it, but the fact that 5 writers independently defaulted to it suggests the rule is not being checked at Tier 1 self-audit.

**Compounding Rule candidate:** Add an explicit checklist item to the script-writer Tier 1 pre-gate that requires grepping the scripts file for the close VO trailing phrase before finalizing.

### Pattern 2 — Tony MCU as hook visual style without a non-Tony third variation
Scripts 3, 4, and 11 all required intervention to ensure Hook C was not also a Tony MCU visual. In Script 4, all three hooks defaulted to Tony MCU. The current criteria say "Hook A/B/C each use a different visual style" but don't call out Tony A-roll as a visual style that consumes all three slots if used twice.

**Compounding Rule candidate:** Add to Video Script Criteria: "If Tony A-roll is used in Hook A or Hook B, Hook C must use a non-Tony visual style (product, graphic, lifestyle, UGC protagonist, or text-only)."

## Promote to Global Criteria

### Candidate 1
**Pattern:** Writers defaulting to "link below" in 5+ scripts per batch without grepping for overuse.
**Exact rule to add:** "Before finalizing the Close, grep the `scripts_file` for `link below` (case-insensitive). If the phrase already appears in 2 or more prior scripts in this file, use `mylifeforce.com` (or equivalent brand URL) as the close CTA instead."
**Target doc:** `00 Global/Criteria/Video Script Criteria.md`
**Suggested insertion point:** Under the Close Types section, after the list of close types, as a new sub-rule: "Close CTA uniqueness (link below cap)."
**Count:** 5 scripts this batch (T004). Check T003 for prior occurrence.

### Candidate 2
**Pattern:** Tony A-roll hook visual style used in 2+ hook variations without an explicit rule preventing it from consuming all three slots.
**Exact rule to add:** "Tony A-roll is a named visual style. If Hook A or Hook B uses Tony A-roll (MCU, face shown), Hook C must use a different visual style (product b-roll, graphic, lifestyle, UGC protagonist, text-only, kinetic text, or split-screen). No two of the three hooks may use Tony MCU as the dominant opening visual."
**Target doc:** `00 Global/Criteria/Video Script Criteria.md`
**Suggested insertion point:** Under Hook Variations section, immediately after "Hook A/B/C each use a different visual style" — as a concrete example of that rule applied to the Tony A-roll case.
**Count:** 3 scripts this batch (T004) required intervention.
