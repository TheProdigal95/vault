# Script Structure: Numbered Reasons

The **Numbered Reasons** (or Listicle) format is a classic direct-response structure that organizes benefits into a scannable, rhythmic sequence. Each "reason" acts as a mini-PAS (Problem-Agitation-Solution) loop, making it highly effective for feature-rich products.

### **Core Breakdown**
- **Structure:** Organization into 3-5 distinct reasons (e.g., "#1 The Fatigue," "#2 The Brain Fog").
- **Time to Benefit:** ~3 seconds. Each reason *is* a benefit reveal.
- **Rhythm:** Rapid pacing between reasons (7-10 seconds per beat). Uses numbered header cards as pattern interrupts.
- **Duration:** 30-50 seconds.
- **Close type:** Ease/simplicity or trust-stacked proof.
