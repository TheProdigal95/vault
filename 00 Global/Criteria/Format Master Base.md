# Format Master Base

Single source of truth for all ad formats/structures used across the agency. Each entry defines the canonical name, core mechanic, sub-variants, production methods, and cross-brand adoption history. Use this when selecting formats for batch planning, delegating to sub-agents, or onboarding a new brand.

Apply on top of [[Universal Copy Rules]], [[Video Script Criteria]], and [[Creative Image Ad Criteria]].

---

## How to Use This Document

- **Batch planning:** Cross-reference with `angles` and `formats` CLI output to match winning angles to proven structures. Each format entry includes verticals tested so you can see what translates.
- **New brands:** Start with formats marked **Universal** — proven across 3+ brands/verticals. These are the safest first-batch selections.
- **Format innovation:** New formats discovered in ads analysis get added here with a provisional entry (mark `Status: Provisional` until tested).
- **Sub-agent dispatch:** When delegating scripts or briefs, reference the canonical format name from this doc in the batch plan. Sub-agents read the Per-Structure Rules here or in [[Video Script Criteria]] to write correctly.
- **Naming:** Always use the **canonical name** from this document in batch plans, working documents, scripts, briefs, and test logs. Aliases are listed for searchability but should not appear in new work.

## Naming Convention

- **Canonical name** is bolded and used in all batch plans, scripts, and working documents
- **Aliases** are listed in parentheses — these are old names that map to the canonical entry. Do not use aliases in new work.
- **Sub-variants** are indented under the parent format with their key differentiator
- **Production method** (green screen, AI avatar, UGC, VO + B-roll, etc.) is a MODIFIER on a format, not a standalone format. "Green Screen Mechanism Explainer" = Mechanism Explainer format + green screen production method.

## Adoption Tiers

| Tier | Definition | Batch Planning Guidance |
|---|---|---|
| **Universal** | Tested across 3+ brands/verticals with documented performance | Safe default for any brand's first batch |
| **Multi-Brand** | Tested across 2 brands | Good candidate for expansion; check vertical fit |
| **Brand-Specific** | Tested in 1 brand only | Test cautiously; may be vertical-dependent |
| **Provisional** | Identified but not yet tested | Add to batch as 1-2 test concepts max |

### V1. Progression Timeline

**Canonical Name:** Progression Timeline
**Aliases:** Benefits timeline, 30-day timeline, milestone progression, temporal discounting timeline
**Tier:** Universal
**Core Mechanic:** Time-stamped milestones (Day 1 → Week 2 → Month 1) showing progressive product benefit. Each milestone names a specific felt outcome paired with its mechanism. Temporal discounting drives urgency — the first benefit appears within hours or days.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Standard benefits timeline | 3-4 milestones, mechanism at each stage | IM8 T001-T005, Elevate T005 |
| Countdown timeline | Reverse chronology ("30 days ago I almost didn't...") | Elevate T005 (#13) |
| Financial timeline | Milestones anchored to cost savings, not health outcomes | Elevate T005 (#12) |
| Joints / mobility timeline | Single-symptom-cluster focus across milestones | IM8 T005 (#6) |
| 60-day extended | Longer horizon for physical products with slower payoff | COW T001 |

**Production Methods:** VO + B-roll (primary), AI avatar over B-roll, UGC talking head with milestone card overlays

**Performance Signals:**
- IM8 T001: Gut Health progression timeline hit #1 top spender on Meta. Temporal discounting + split-screen CGI drove perceived immediacy.
- IM8 T005: Timeline format at 37-42% D7 ROAS in menopause campaign (vs. 100-121% for emotional testimonial in same campaign — format signal, not angle signal).
- Stellar Sleep T002: 4-week progress timeline tested as static (#15).

**Cross-Brand Adoption:** IM8 (T001-T005), Elevate (T005), COW (T001), Stellar Sleep (T002 static). Migrated from IM8 → Elevate in T005 with emotional and financial sub-variants.

**Per-Structure Rules:** Codified in [[Video Script Criteria]] → Per-Structure Rules. Key rules:
- 3-4 milestones max; each names the ingredient/mechanism
- Honest starts ("first sip" energy/joints, "first night" sleep, "Day 14" gut)
- Use days not weeks past month 1
- Hook opens with "From your first sip/night/day" or similar immediate-benefit framing
- Duration ~45-60s

**Verticals Tested:** Health/wellness supplements (IM8), medical weight loss (Elevate), sleep apps (Stellar Sleep), footwear (COW)

### V2. Mechanism Explainer

**Canonical Name:** Mechanism Explainer
**Aliases:** Educational mechanism + CGI, Green Screen Educational, Educational / Reframe, Educational / Authority Lead, AI Nurse Greenscreen / Mechanism, Educational Explainer, Mechanism + Gut Explainer, Disqualifier / Reframe, Whiteboard
**Tier:** Universal
**Core Mechanic:** Educates the viewer on HOW or WHY the product works at a biological, technical, or structural level. The mechanism IS the selling — understanding it creates the "aha" that converts. Distinguishes from Talking Head by the educational architecture: the video is structured to teach a pathway, not just deliver opinions.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Green screen over CGI/B-roll | Talent composited over mechanism visualizations | IM8 T001-T005, FitSleeps T001 |
| Whiteboard / disqualification | Creator disqualifies alternatives (red X) then reveals the correct pathway | IM8 T002, Stellar Sleep T002 |
| AI character explainer | AI-generated character delivers the education | IM8 T003-T005 |
| VO + B-roll explainer | No on-camera talent; narration over diagrams and cutaways | Stellar Sleep T002 |
| Four Nutrients structure | Specific sub-variant: 4 named nutrients each paired with a mechanism | IM8 T004-T005 |
| Anxiety Feedback Loop | Mechanism-as-diagnosis: names a loop the viewer is stuck in | Stellar Sleep T002 (#3) |
| Nurse AI greenscreen | AI nurse character over foot anatomy CGI | COW T001 |

**Production Methods:** Green screen (primary), AI animated character, VO + B-roll, creator on camera with overlays

**Performance Signals:**
- IM8 T005: Mechanism/numbered-reasons format at 37-42% D7 ROAS in menopause (strong spend but lower ROAS than emotional formats in same campaign).
- IM8 T002: "Dr. James" authority educational doing well on AppLovin — authority resonates.
- IM8 T005: Four Nutrients structure = $6,811 top spender with 4.73-6.50% CTR conditional hook.
- FitSleeps T001: Green screen mechanism education (sound habituation vs. wrist vibration) — new for FitSleeps, tests a single clear mechanism.

**Cross-Brand Adoption:** IM8 (T001-T005), FitSleeps (T001), Stellar Sleep (T002), COW (T001), Elevate (T004-T005). Present in every brand. Most versatile format.

**Per-Structure Rules:** Partially codified in [[Video Script Criteria]] under "Talking-head (green screen / whiteboard)" and "Green screen educational." Combined and expanded rules:
1. Education before product — mechanism reveal earns the right to sell. Product appears after the viewer understands the pathway.
2. Creator enters within 5s. CGI functions as pattern interrupt, not decoration.
3. Alternate full-frame and corner-box every 5-8s to maintain visual variety.
4. Disqualify alternatives before revealing the mechanism (red X on failed solutions → circle on correct pathway).
5. Creator = knowledgeable friend, not doctor. Name the doctor/authority, reference them — don't claim to be one.
6. B-roll palette: labels, nutrient tables, product shots, ingredient close-ups, certifications, lifestyle outcomes.
7. Faster product reveal than general educational (~0:15 for green screen variant vs. ~0:30 for full educational).
8. Max 2 CGI cutaways per body as pattern interrupts — not every mechanism step needs CGI.
9. Duration: ~30-50s (green screen), ~30-50s (whiteboard).
10. The mechanism explanation must connect to a felt experience at the end — pure education without a benefit beat reads as a lecture.

**Verticals Tested:** Health/wellness supplements, sleep tech, sleep apps, footwear, medical weight loss

### V3. Talking Head

**Canonical Name:** Talking Head
**Aliases:** UGC talking-head direct to camera, Direct to camera, Creator on camera
**Tier:** Universal
**Core Mechanic:** Talent speaks directly to camera delivering the message. The person IS the ad — their credibility, relatability, or authority carries the conversion. Distinguishes from Mechanism Explainer by structure: Talking Head is opinion/story-driven, not education-driven.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Standard UGC talking head | Single creator, direct to camera, personal testimony | FitSleeps T001, Elevate T004 |
| Talking head + persistent overlays | Direct to camera with persistent banner/price/logo overlays | Elevate T005 (#6, #8) |
| AI talking head | AI-generated avatar delivering the message | Elevate T004-T005, IM8 T004-T005 |

**Production Methods:** UGC creator (primary), AI-generated avatar, in-house talent

**Performance Signals:**
- Elevate T004: 7 of 15 scripts used talking head — primary format for character voice testing.
- FitSleeps T001: UGC talking-head ("I tried everything") tested as new format for FitSleeps.

**Cross-Brand Adoption:** FitSleeps (T001), Elevate (T004-T005), IM8 (T004-T005). Universal production method across all brands.

**Per-Structure Rules:** Codified in [[Video Script Criteria]] → Per-Structure Rules under "Talking-head (green screen / whiteboard)." Key rules:
- CGI as pattern interrupt; creator enters within 5s
- Alternate full-frame and corner-box every 5-8s
- Duration ~30-50s
- Creator references authority, doesn't claim to be one

**Verticals Tested:** Sleep tech, medical weight loss, health/wellness supplements

### V4. Narrative Testimonial

**Canonical Name:** Narrative Testimonial
**Aliases:** UGC / story with narrative thread, First-person narrative UGC-style, Full narrative arc, Problem-solution narrative, Authority / story
**Tier:** Universal
**Core Mechanic:** First-person story with a complete arc — problem → discovery → transformation. The narrative itself IS the proof. Each clip or beat serves the arc, not a montage of unconnected moments. Two sub-categories in [[Video Script Criteria]]: story-driven (authority/story) and testimonial-driven (UGC narrative thread).

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Authority / story testimonial | Story earns the delay; real details, named authority, documentary tone | IM8 T001-T005 |
| UGC narrative thread | Overarching narrative across clips — NOT a quote montage | IM8 T001-T005, FitSleeps T001, Elevate T004 |
| Full narrative arc | Extended emotional arc (40-60s), tension → skepticism → proof | IM8 T005 (#15, #18) |
| AI narrator testimonial | AI voice narrates a composite first-person story over B-roll | IM8 T003-T005, Stellar Sleep T002 |

**Production Methods:** UGC creator, AI narrator VO + B-roll, AI avatar, podcast interview setting

**Performance Signals:**
- IM8 T005: Emotional testimonial format at 100-121% D7 ROAS in menopause campaign — 2.5-3x higher ROAS than mechanism formats in the same campaign, same audience, same LP.
- IM8 T005: Menopause Sleep Hot Flashes testimonial $3,261 spend at 100% D7 ROAS; Menopause Testimonials_09 at $1,605 spend, 121% D7 ROAS.
- IM8 T005: Fitness Muscle Recovery testimonial at 571% D7 ROAS on feel-again LP ($83 spend).
- Lifeforce: Ashley T UGC testimonial = best CTR (15.76%) AND second-best ROAS (0.290) at lowest CPM ($25.81). Single best-performing creative.

**Cross-Brand Adoption:** IM8 (T001-T005), FitSleeps (T001), Elevate (T004), Lifeforce (top spender). Strongest conversion format in the agency.

**Per-Structure Rules:** Codified in [[Video Script Criteria]] → Per-Structure Rules under "Authority / story" and "UGC / story with narrative thread." Key rules:
- Authority/story: Narrative arc with tension (skepticism, risk, proof) — not endorsement. Real details. Max 2 transformation claims per 8s section. Story earns the 15-20s delay to benefit. Duration 40-60s.
- UGC narrative: Overarching narrative, NOT a quote montage. Each clip serves the arc. Intercut 2-3s B-roll. Duration ~35-50s.

**Verticals Tested:** Health/wellness supplements, sleep tech, medical weight loss, longevity/TRT

### V5. Numbered Reasons / Listicle

**Canonical Name:** Numbered Reasons / Listicle
**Aliases:** Numbered reasons, Listicle video, Reasons why video
**Tier:** Multi-Brand
**Core Mechanic:** Counted list where each reason runs a mini-PAS cycle (problem → product's solve). The number in the hook creates a commitment loop — viewer stays to see all items. Benefits start at ~3s (each reason IS a benefit).

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Standard numbered reasons | 3-5 reasons, each a mini-PAS | IM8 T001-T005 |
| Tier list | Ranked comparison using S/A/B/C tier format | Stellar Sleep T001, FitSleeps T001 |
| Rapid-fire kinetic cards | Compressed numbered cards as mechanism flash (no VO) | IM8 T005 (#19, #20 hybrid) |

**Production Methods:** Creator on camera, AI avatar, kinetic text over B-roll, green screen

**Performance Signals:**
- IM8 T005: Numbered-reasons at 37-42% D7 ROAS (mechanism cluster). Strong spend, lower conversion than testimonial.

**Cross-Brand Adoption:** IM8 (T001-T005), FitSleeps (T001 tier list variant), Stellar Sleep (T001 tier list). Tested across health and sleep verticals.

**Per-Structure Rules:** Codified in [[Video Script Criteria]] → Per-Structure Rules. Key rules:
- Each reason runs mini-PAS (problem → product's solve)
- Benefits start ~3s
- Close with ease/simplicity
- Product reveal can be late
- Duration 30-50s

**Verticals Tested:** Health/wellness supplements, sleep tech

### V6. Selfie Walk + Trending Audio

**Canonical Name:** Selfie Walk + Trending Audio
**Aliases:** Shame to Pride Selfie Walk, Text wall selfie walk
**Tier:** Brand-Specific (Elevate) — candidate for Multi-Brand expansion
**Core Mechanic:** Single continuous selfie walk shot with persistent on-screen text captions stacking throughout. Trending IG audio carries the sound-on experience. Captions carry the sound-off experience. The format converts because the combination of raw vulnerability + aspirational audio + text density filters for high-intent viewers.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Shame to pride arc | Emotional progression from vulnerability to confidence | Elevate T002 (original), T005 (#1-#5) |
| Price-hero selfie walk | $8/day or $58/week as the dominant text element | Elevate T005 (Hook D variants) |
| Identity recovery | Pure aspiration — no struggle, only the desired state | Elevate T005 (#5) |

**Production Methods:** UGC creator (filmed selfie walk), AI-generated avatar walk

**Performance Signals:**
- Elevate T002: Best video format. "Shame-to-pride arc + text wall + aspirational b-roll + trending audio" confirmed as best video format for Elevate.
- Elevate T005: 5 of 14 scripts use selfie walk — scaled as Track A.

**Cross-Brand Adoption:** Elevate only (T002, T005). Strong candidate for GLP-1 and wellness verticals where shame/identity transformation is a core pain.

**Per-Structure Rules:** Codified in [[Video Script Criteria]] → Per-Structure Rules. Key rules:
- Single continuous selfie walk, no cuts
- Persistent on-screen captions throughout (each new line appears as previous stays or fades)
- Only hooks vary across A/B/C — body captions and walk footage identical
- One body row with all captions as `<br><br>` block
- No separate Close section (fold CTA into body captions)
- Duration ~15s
- Caption IS the hook — sound-off first; trending audio carries sound-on

**Verticals Tested:** Medical weight loss

### V7. AI Animated Character

**Canonical Name:** AI Animated Character
**Aliases:** AI avatar, AI character VSL, 3D character narrator
**Tier:** Multi-Brand
**Core Mechanic:** AI-generated character composited over B-roll from 0:00. The character functions as a relatable narrator — establishing persona in the hook, then alternating between full-frame and corner-box to maintain visual variety. Body B-roll fills the educational or emotional content.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| AI nurse character | Nurse persona for healthcare/shift-worker targeting | COW T001, IM8 T003-T005 |
| AI narrator (general) | Non-specific persona narrating over mechanism B-roll | IM8 T003-T005, Stellar Sleep T001-T002 |
| 3D character + chart | AI character pointing to/building comparison charts | Stellar Sleep T002 (#7) |

**Production Methods:** AI-generated avatar (primary), composited over CGI/B-roll/stock footage

**Performance Signals:**
- IM8 T002: Dr. James AI avatar doing well on AppLovin.
- Stellar Sleep T001-T002: AI characters used for all video production (AI-only production confirmed).

**Cross-Brand Adoption:** IM8 (T003-T005), COW (T001), Stellar Sleep (T001-T002). Used wherever UGC creators are unavailable or AI-only production is required.

**Per-Structure Rules:** Codified in [[Video Script Criteria]] → Per-Structure Rules. Key rules:
- Character composited over B-roll from 0:00; establish persona in hook
- Alternate full-frame and corner-box every 5-8s
- ≥3 visual layers per body beat of ~10s (character frame + B-roll + overlay/CGI)
- Body B-roll = labels, nutrient tables, product, certifications
- 2-3 symptoms max (AppLovin 60s cap)
- Duration 30-60s

**Verticals Tested:** Health/wellness supplements, sleep apps, footwear

### V8. Comparison / Elimination

**Canonical Name:** Comparison / Elimination
**Aliases:** Visual Crossout, Stack Replacement, Ranked Comparison, Side-by-Side Comparison, Authority Rated Elimination, Caption-only Split Comparison
**Tier:** Multi-Brand
**Core Mechanic:** Sequential or simultaneous comparison that disqualifies alternatives and positions the product as the winner. The viewer watches options get eliminated — creating a "last one standing" effect. Can be structured as a ranked ladder, a side-by-side table, or a serial dismissal.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Ranked comparison | Competitors ranked worst-to-best, product wins | COW T001, Stellar Sleep T001 |
| Visual crossout / stack replacement | Failed alternatives shown and crossed out, replaced by product | IM8 T001-T005 |
| Side-by-side split | Two options shown simultaneously, one clearly better | Elevate T005 |
| Authority rated elimination | Expert/authority figure rates and eliminates options | COW T001 |
| Animated comparison chart | Static comparison table animated into video | Stellar Sleep T002 (#7) |

**Production Methods:** VO + B-roll with overlay graphics (primary), AI avatar, green screen, kinetic text

**Performance Signals:**
- Lifeforce: Comparison image format = best ROAS in account (0.321). The format that actually converts.
- COW T001: Ranked comparison (3 competitors → ComfortWear wins) modeled on the format spending $70K in account.
- COW T003: Three-problem debunking kept as core format but evolving the problem set.

**Cross-Brand Adoption:** COW (T001-T003), IM8 (T001-T005), Stellar Sleep (T001-T002), Elevate (T005), Lifeforce (top spender as static). Translates across all verticals where competitors or failed alternatives exist.

**Per-Structure Rules (NEW — not yet in Video Script Criteria):**
1. Name solution categories, never competitor brand names — unless the brand's account already uses competitor names (COW uses Hoka, Skechers by name; blur logos in footage).
2. Minimum 3 alternatives disqualified before product reveal. Each gets a specific failure reason (mechanism-level, not just "didn't work").
3. Product reveal timing: alternatives are dismissed in the first 60% of the video; product takes the final 40%.
4. Visual cue for elimination is mandatory — red X, crossout, fade-to-grey, swipe away. The viewer must SEE the rejection, not just hear it.
5. The winning product must be positioned on a different axis than the eliminated options. If all alternatives failed on "symptom relief," the product wins on "root cause" — not just "better symptom relief."
6. Duration: 30-50s. Tighter than educational formats because the comparison structure creates inherent momentum.

**Verticals Tested:** Health/wellness supplements, footwear, sleep apps, medical weight loss, longevity/TRT

### V9. Text Wall Video

**Canonical Name:** Text Wall Video
**Aliases:** Text Wall 2.0, Text overlay video, Caption-only video
**Tier:** Multi-Brand
**Core Mechanic:** Dense on-screen text over moving B-roll or static imagery. The text IS the ad — no voiceover, or minimal VO. Filters for high-intent viewers through copy density. Works because scroll-stopping text volume + emotional/visceral B-roll creates a "can't look away" effect that self-selects buyers.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Static background text wall | Text over static imagery (original) | Stellar Sleep T001 |
| Text Wall 2.0 (moving B-roll) | Text over cinematic AI B-roll with subtle motion | Stellar Sleep T002 |
| Selfie walk text wall | Text stacking over continuous selfie walk footage | Elevate T002-T005 |
| Pain-stack text wall | VoC-heavy pain language dominating the text | Stellar Sleep T002 (#1) |

**Production Methods:** AI B-roll + text overlays (primary), UGC selfie walk + text, stock footage + text

**Performance Signals:**
- Stellar Sleep T001: Text walls = top spenders. The format filters for high-intent users through VoC density.
- Stellar Sleep T001: "Founded at Harvard" hero card hit 5.3% CTR — highest CTR driver — because it sells access to a "valuable secret."
- Elevate T002: Selfie walk variant (which is a text wall hybrid) confirmed as best video format.

**Cross-Brand Adoption:** Stellar Sleep (T001-T002), Elevate (T002-T005 as selfie walk variant), COW (T001). Migrated from Stellar Sleep → Elevate with the selfie walk production method.

**Per-Structure Rules (NEW — not yet in Video Script Criteria):**
1. Every text card must work as a standalone headline on mute. Apply [[Headline & Text Hook Criteria]].
2. Text density is the feature, not a bug — but each card must earn its screen time. Cut any card that doesn't add new information or emotion.
3. No VO by default. If VO is added (Text Wall 2.0), it provides a parallel channel — not a reading of the on-screen text.
4. B-roll must be emotionally aligned with the text content at each beat. 3AM clock → exhaustion text. Morning sunlight → relief text.
5. 5 text overlay variations per concept (each roughly the same character length) count as a single video concept for production but test different angles.
6. Duration: 15-30s. Text walls are short — density over duration.

**Verticals Tested:** Sleep apps, medical weight loss, footwear

### V10. VO + B-Roll (No Talking Head)

**Canonical Name:** VO + B-Roll (No Talking Head)
**Aliases:** Narrator over B-roll, AI narrator + lifestyle, Pure B-roll with narration
**Tier:** Multi-Brand
**Core Mechanic:** Voiceover narration over lifestyle, product, or CGI B-roll with no on-camera talent visible. The VO carries the persuasion; the B-roll carries the visual engagement. Distinguishes from Talking Head by the absence of on-camera talent and from Mechanism Explainer by being narrative/emotional rather than educational.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Standard VO + lifestyle B-roll | Narration over lifestyle scenes matching the pain/benefit | Elevate T004-T005, IM8 T005 |
| VO + persistent overlays | Narration with "This Failed You" / price banners persisting | Elevate T005 (#7, #10) |
| VO + CGI B-roll | Narration over mechanism animation / CGI cutaways | IM8 T003-T005 |

**Production Methods:** AI narrator VO (primary), recorded VO, stock B-roll, AI-generated B-roll, CGI

**Performance Signals:**
- Elevate T004: 3 of 15 scripts used VO + B-roll. Tested as complement to talking head and green screen.
- Elevate T005: VO + B-roll used for blame framing concepts (#7, #10) with persistent overlay treatment.

**Cross-Brand Adoption:** Elevate (T004-T005), IM8 (T003-T005), Stellar Sleep (T002). Growing format as AI narration quality improves.

**Per-Structure Rules (NEW — not yet in Video Script Criteria):**
1. VO carries the ad — every VO line must stand alone as persuasive copy, not just describe what's on screen.
2. Visual column must name a new shot or cutaway roughly every 2-4s. Generic "lifestyle b-roll" placeholders over long VO beats are a flag. See Visual & B-Roll Criteria rule 13 in [[Video Script Criteria]].
3. 2-3 distinct visual modalities across the script (lifestyle B-roll + product shots + CGI/overlay minimum). Pure single-modality loses engagement.
4. Persistent overlays (banners, payment logos, price cards) are a modifier — when used, they create a "template" feel that should be consistent throughout, not added mid-video.
5. Color grade shift from problem (cool/muted) to solution (warm/natural) is the default emotional mapping.
6. Duration: 30-45s. Slightly shorter than talking head because there's no on-camera talent creating parasocial hold time.

**Verticals Tested:** Medical weight loss, health/wellness supplements, sleep apps

### V11. Self-Evaluation NO VO

**Canonical Name:** Self-Evaluation NO VO
**Aliases:** Self-diagnosis checklist, Symptom checklist video, Self-ID quiz format
**Tier:** Multi-Brand
**Core Mechanic:** Checklist or self-diagnosis cards stack on screen over lifestyle B-roll with music only — no voiceover. The viewer self-identifies by counting how many items apply to them. The format creates personal stakes through self-assessment rather than external persuasion. Highest CTR format in the agency.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Symptom checklist | "How many of these do you have right now?" | IM8 T005 (#3, #8) |
| Barrier checklist | "How many of these sound like you?" — access barriers, not symptoms | Elevate T005 (#14) |
| Lifestyle checklist | Daily-life friction points as checklist items | IM8 T005 (#16 — energy persona) |

**Production Methods:** Music + text overlays over lifestyle B-roll (primary), kinetic text cards, AI B-roll

- IM8 T005: Self-evaluation NO VO = 5.36-5.99% CTR — highest CTR format in the IM8 account. Underscaled at time of T005 planning.
- IM8 T005: Format multiplied across 3 personas (gut health, menopause, energy) to test persona generalizability.
- Elevate T005: Adapted for medical weight loss as "barrier checklist" variant — access/cost barriers instead of health symptoms.

**Cross-Brand Adoption:** IM8 (T005), Elevate (T005). High-value format for expansion to any brand with identifiable symptom clusters or barrier lists.

**Per-Structure Rules (NEW — not yet in Video Script Criteria):**
1. No voiceover — music drives pacing. Trending IG audio or upbeat instrumental that syncs with card appearance rhythm.
2. Hook card must frame the self-evaluation question explicitly: "How many of these [have you tried / do you have / sound like you]?"
3. Each checklist item gets 3-4 seconds. Items should escalate in specificity and emotional weight — start with common symptoms, end with the one that makes them feel truly seen.
4. 5-8 checklist items maximum. Fewer than 5 doesn't create enough self-identification; more than 8 loses momentum.
5. Pivot card after the checklist: one card that reframes the problem ("What if the problem was never willpower?"). This is the format's conversion mechanism.
6. CTA cards follow immediately after the pivot — price, payment options, action prompt.
7. Each checklist card must work as a standalone scan item — no card should require reading the previous card to make sense.
8. Duration: 25-35s. Quick-rhythm format; don't pad.

**Verticals Tested:** Health/wellness supplements, medical weight loss

**Cross-Brand Adoption:** IM8 (AI-generated statics), Stellar Sleep (3D character concepts). Primarily used in brands with AI-only production.

**Verticals Tested:** Health/wellness supplements, sleep apps

### S13. Authority Static

**Canonical Name:** Authority Static
**Aliases:** Study screenshot static, Institutional authority card, Clinical proof static
**Tier:** Multi-Brand
**Core Mechanic:** Screenshot or stylized render of a clinical study, institution logo, expert quote, or certification. The authority element IS the ad — the viewer's trust in the institution transfers to the product.

**Cross-Brand Adoption:** Stellar Sleep (T001-T002 — Harvard study screenshot), IM8 (Dr. James authority), Lifeforce (Tony Robbins). Works for any brand with credible institutional or expert backing.

**Performance Signals:**
- Stellar Sleep T001: "Founded at Harvard" hero card = 5.3% CTR. Highest CTR driver because it sells access to a "valuable secret."
- Stellar Sleep T002 (#13): Harvard Study Screenshot upgraded for T002.

**Verticals Tested:** Sleep apps, health/wellness supplements, longevity/TRT

### S14. Social / Style Static

**Canonical Name:** Social / Style Static
**Aliases:** Lifestyle identity static, Contextual product static
**Tier:** Brand-Specific
**Core Mechanic:** Lifestyle/identity image where the product is contextual, not hero. The viewer sees themselves in the lifestyle first, then notices the product. Identity callout drives the conversion.

**Cross-Brand Adoption:** COW (nurse lifestyle), Lifeforce (male identity/performance). Works for apparel, accessories, and lifestyle-adjacent products.

**Verticals Tested:** Footwear, longevity/TRT

### S15. Native Image + Long-Form Story

**Canonical Name:** Native Image + Long-Form Story
**Aliases:** Long-form primary text ad, Story-driven native image, Whitelisted native photo
**Tier:** Multi-Brand
**Core Mechanic:** Native-looking photo (looks like someone snapped it with their phone) + long-form narrative in the primary text field (1,000-2,200 words). The image stops the scroll by looking like organic content; the primary text tells a complete 13-beat story that sells. The combination filters for high-intent buyers who read the entire story.

**Sub-Variants:**

| Sub-Variant | Key Difference | Brands / Batches |
|---|---|---|
| Standard 13-beat story | Full narrative arc (1,500-2,200 words) | FitSleeps T001 (3 concepts) |
| Shortened story (~1,000 words) | Compressed arc for higher-price products | FitSleeps T001 (#7), Lifeforce T002 |
| Tweet + long-form primary | Tweet-style image paired with long-form text underneath | Lifeforce T002 (~11 concepts) |

**Cross-Brand Adoption:** FitSleeps (T001 — workhorse format, 4 of top 10 spenders), Lifeforce (T002), Elevate (T003 — tested but native photo whitelist didn't convert).

**Performance Signals:**
- FitSleeps: Native image + long-form story = 4 of 10 top spenders. The workhorse format.
- Lifeforce T002: Tweet scroll-stop + FitSleeps-style long-form text = the core format hypothesis (11 of 15 concepts).
- Elevate T003: "Native photo whitelist gets massive engagement and spend but doesn't convert. Killed. High engagement does not equal purchase intent for this format." — Format failed for medical weight loss.

**Verticals Tested:** Sleep tech (strong), longevity/TRT (testing), medical weight loss (failed)

# Production Method Modifiers

Production methods are NOT standalone formats. They modify any format above. When naming a concept in batch plans, use the format name + production method: "Mechanism Explainer (green screen)" or "Narrative Testimonial (AI narrator)."

| Production Method | What It Is | Best Paired With |
|---|---|---|
| **UGC Creator** | Real person filming on their phone/in their space | Talking Head, Narrative Testimonial, Selfie Walk, Podcast Testimonial |
| **AI Avatar / Animated Character** | AI-generated person composited over B-roll | Mechanism Explainer, Talking Head, AI Animated Character |
| **Green Screen** | Talent composited over CGI/B-roll backgrounds | Mechanism Explainer, Comparison / Elimination, Blame Framing |
| **VO + B-Roll** | Narration (AI or recorded) over stock/lifestyle/CGI footage | Any format — this is the most flexible production method |
| **Kinetic Text / Caption Only** | Text cards / overlays with no VO, music only | Text Wall Video, Self-Evaluation NO VO, Selfie Walk |
| **AI-Generated Imagery** | AI-created images for statics or B-roll | Any static format, B-roll for any video format |
| **Screen Recording** | Actual app/platform UI captured | Screen Recording / Native Proof |

# Cross-Reference: Green Screen Is a Production Method

"Green Screen" appears in many batch plans as a format name. It is not a format — it is a production method that modifies other formats:

| What People Call It | What It Actually Is |
|---|---|
| Green Screen Educational | **Mechanism Explainer** (green screen production) |
| AI Nurse Greenscreen | **AI Animated Character** + **Mechanism Explainer** (green screen production) |
| Green Screen Comparison | **Comparison / Elimination** (green screen production) |
| Green screen testimonial | **Talking Head** (green screen production) |

Always use the format name in batch plans and working documents. Note the production method in the visual direction or batch plan's "Visuals" column.
