# Hook Quality

Scores whether the hook earns the scroll-stop: grounded in buyer language, creates specific curiosity, fits the persona, and deploys heuristics in the first 5 seconds.

---

## What This Scores

The hook is the most leveraged 5 seconds in the script. A technically correct script with a weak hook never gets watched. This dimension evaluates: does the hook use real buyer language (not marketing copy), create a specific curiosity gap, interrupt the viewer's pattern, fit the target persona, and deploy 1-2 psychological heuristics in the opening seconds? Hooks that don't trace to winning ads analysis or VoC language are first-draft failures.

## Calibration Table

| Score Range | Description |
|---|---|
| 90-100 | Hook language traces to winning hook transcriptions AND/OR verbatim VoC phrases (source named). Pattern interrupt in first 2 seconds. Specific curiosity gap (not vague). Persona's pain/trigger named before hook ends. All 3 variations use different core types for captions AND different visual styles. 1-2 heuristics firing in the first 5 seconds (correctly identified). |
| 70-89 | Hook is competent but generated from first principles — language sounds like good marketing copy but doesn't mirror buyer language. Curiosity gap exists but is generic. OR: all 3 hook captions use the same approach (3 reframes, 3 stats). OR: heuristics present but misidentified (e.g., milestone timeline labeled "temporal discounting"). |
| 50-69 | Hook is generic, could apply to any brand in the category. No traceable connection to winning hooks or VoC. Curiosity gap is vague or absent. Pattern interrupt relies on common tactics without specificity. |
| Below 50 | Hook generated entirely from scratch with no account data reference. Uses demographic callouts instead of pain-triggers. No pattern interrupt. Duplicates another hook in the batch. Rewrite from scratch — evaluator must name the specific VoC phrase and winning hook pattern to use. |

## Instant-Fail Triggers

- **Zero traceable connection to winning ads or VoC.** Auto-caps at 49.
- **Demographic callout as primary hook mechanism.** "Busy moms," "men over 40," "women in their 30s." Auto-caps at 49.
- **Hook duplicates another hook in the batch.** Same Curiosity Gap angle, same Reframe payoff, or same Provocation. Auto-caps at 49.

---

## Scoring Checklist

- **Account data grounding:** Can the hook VO/caption be traced to a specific winning hook from ads analysis (name the ad) or a specific VoC phrase from reviews/research (quote the phrase)?
- **Pattern interrupt:** Does something in the first 2 seconds break the scroll pattern — visual, verbal, or structural?
- **Curiosity gap specificity:** Is the gap about something specific the viewer wants to know? Would someone with zero context understand the problem AND feel curiosity?
- **Persona fit:** Does the hook name the persona's condition or pain-trigger before it ends? In VO or caption?
- **Caption variety:** Do the 3 hook captions use different core types per Headline & Text Hook Criteria? Not 3 variations of the same type.
- **Visual style distribution:** Do Hook A/B/C each use a different visual style?
- **Heuristic deployment:** Are 1-2 heuristics firing in the first 5 seconds? Name them. Are they correctly identified? (Temporal discounting = immediate results like "from your first sip," NOT milestone timelines. Framing effect = reframed numbers. Affect heuristic = dropping INTO an experience.)
- **Problem-first bridge:** If the hook is problem-first, does it hint a solution is coming?
- **All hooks have VO:** No hook is caption-only when the body has voiceover.

---

## Feedback Format

When scoring below 90:
- Which of the checklist items failed
- The current hook text (VO + caption for the failing hook variation)
- What account data or VoC phrase it should trace to (exact source ad name or exact review quote)
- A concrete rewrite direction using that source material — not "make it more grounded" but "rewrite Hook A VO using the winning hook pattern from Ad 03 ('[exact phrase]') combined with VoC phrase '[exact phrase]'"
- If heuristic deployment failed: which heuristic was misidentified or missing, with the correct definition from the Heuristic Reference
