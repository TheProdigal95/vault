### Caption-VO Echo

Captions and VO are two channels delivering two messages. When both say the same thing, the viewer reads data they're already hearing — no added value, no engagement.

| Severity | Finding | Affected Scripts | Linked Rule | Status |
|---|---|---|---|---|
| ❌ | Hook C: VO says "95% of users felt more energy in a 12-week clinical trial." Caption says "12-WEEK CLINICAL TRIAL / 95% OF USERS FELT MORE ENERGY." Same content on both channels. | #3 Recovery Reframe | On-Screen Caption Variety; Video Script Criteria — Caption ≠ VO | Fixed |

### Data Balance — All 3 Channels Delivering Data Simultaneously

When VO delivers data, captions echo the same data, AND visuals show a static product flat-lay with stat overlay, the first 7 seconds read as a data dump. At least one channel must carry emotional or visual engagement.

| Severity | Finding | Affected Scripts | Linked Rule | Status |
|---|---|---|---|---|
| ❌ | Hook C: VO = clinical trial stat. Caption = same stat. Visual = product flat-lay with stat overlay. Zero emotional engagement in the first 7 seconds. All three channels are facts. | #3 Recovery Reframe | Data Balance Rule | Fixed |

**Proposed fix for Script #3 Hook C:** Rewrite the caption to something winner-inspired (e.g., "YOUR RECOVERY ISN'T SLOWING DOWN — IT'S UNDER-FUELED" or "WHAT HAPPENS WHEN YOU FILL THE GAPS"). Replace the static product-with-stat visual with something engaging — person stretching in gym, CGI nutrient flow, or dynamic B-roll. Keep the stat VO if needed, but let the other two channels add engagement rather than echo data.

## Cross-Batch Patterns — Promote to Global Criteria

### Pattern 1: IM8 absent from body (5 scripts + 3 borderline = 8/15)

More than half the batch had the product name missing from the body or pushed to the very last beat. This isn't a one-off — it's a systematic gap in how mechanism explainers and timelines get written. The writer focuses on ingredient names and forgets to anchor them to the product.

**Promotion candidate:** The strengthened Early Product Mention rule was already added to the Working Document's Script Criteria during this review. Candidate for promotion to `Video Script Criteria.md` global checklist if it recurs in T004.

### Pattern 2: Hook captions default to problem-only (7/15 scripts)

Nearly half the batch had all 3 hook captions focused on the same angle — problem statements, reframes, or invalidation. None of the failing scripts included a result promise, curiosity promise, or stat-first caption. The winning top spenders (Ad 03: "WHAT HAPPENS TO YOUR GUT IN 90 DAYS", Ad 02: hook 70) use clear promise captions, not just problem naming.

**Promotion candidate:** The On-Screen Caption Variety rule was already added to the Working Document. If this pattern recurs across another batch, promote to `Headline & Text Hook Criteria.md`.

### Pattern 3: Visual column readability (15/15 scripts)

100% of scripts have clumped visual descriptions. This is a formatting convention gap, not a creative quality issue. One formatting pass would fix it batch-wide.

**Promotion candidate:** Already added to Working Document. If adopted, add to `Video Script Criteria.md` as a formatting requirement for the Visual column.
